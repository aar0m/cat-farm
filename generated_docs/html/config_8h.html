◆ ANIMAL_FARM_RNG

#define ANIMAL_FARM_RNG   std::mt19937

Establish the RNG used by Animal Farm.

The C++ standard library has some great built-in random number features.

See also
https://en.cppreference.com/w/cpp/header/random

If you want the same pseudorandom sequence each time you run the program, use std::mt19937.

If you want a truly unique random sequence each time you run the program, use std::random_device.
