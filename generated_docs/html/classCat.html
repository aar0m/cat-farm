Cat::Cat ( std::string_view  newName)
explicit

Generates a Cat with an unknown 'color', Gender, random 'weight', and specified 'newName'.

This constructor is declared to be explicit, so you can't do silly things like Cat newCat = "Bella"; ... instead, you need to construct a Cat properly like Cat newCat( "Bella" );
